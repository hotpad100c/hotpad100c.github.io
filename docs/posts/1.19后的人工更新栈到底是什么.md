> 栈溢出更新抑制器在1.19.2被修复。这期间，Minecraft引入了一套新的更新调度系统。
虽然这个系统尽力模仿了更新的行为，但仍然存在严重的行为变动。由于原版红石元件的限制，
这些问题没有过多体现，本文旨在分析新的更新系统，并指出其带来的问题。

# 更新的实体化

在此之前，我们一直将“方块更新”理解为 世界（Level） 对方块更新方法的**一系列调用**。
在这种理解中，方块更新并不是一个具有形式的“实体”，而只是由方法调用组成的传播过程。

然而，要修复连锁更新导致的栈溢出，必须赋予这种原本无形的更新以“实体”，使其能被计数、管理与调度。

因此，现在的方块更新被分为了以下4类基本单位：

---
## 单次 形状更新（PP更新）

```java

private record ShapeUpdate(...) implements NeighborUpdates {
    @Override
    public boolean runNext(final Level level) {
        NeighborUpdater.executeShapeUpdate(...);
        return false;  
    }
}
```

PP更新的唯一单位，仅代表一次PP更新，所有PP更新都使用这个单位。
例如，一般围绕“更新核”发出PP更新时，分别是6个独立的`ShapeUpdate`。

## 单次 邻居更新（NC更新）

```java
private record SimpleNeighborUpdate(...) implements NeighborUpdates {
    @Override
    public boolean runNext(final Level level) {
        BlockState state = level.getBlockState(this.pos);
        NeighborUpdater.executeUpdate(...);
        return false;  
    }
}
```

NC更新的最小单位，仅代表一次NC更新。
比如侦测器，中继器等所指向的那一格，比较器更新发出的更新就是这个。
值得注意的是与PP更新不同，围绕“更新核”发出NC更新时，不使用这个单位。

## 六向 邻居更新（NC更新）

```java
private record FullNeighborUpdate(...) implements NeighborUpdates {
    @Override
    public boolean runNext(final Level level) {
        NeighborUpdater.executeUpdate(...);
        return false;
    }
}
```
NC更新的集体单位，代表六次NC更新。
一般围绕“更新核”发出NC更新时，使用这个单位。

## 五向 邻居更新（NC更新）

```java
private static final class MultiNeighborUpdate implements NeighborUpdates {
    private int idx = 0;
    
    @Override
    public boolean runNext(final Level level) {
        Direction direction = NeighborUpdater.UPDATE_ORDER[this.idx++];
        // ...略...
        NeighborUpdater.executeUpdate(...);
        if (this.idx < NeighborUpdater.UPDATE_ORDER.length && NeighborUpdater.UPDATE_ORDER[this.idx] == this.skipDirection) {
            this.idx++; //五向：剔除某个方向的更新
        }
        return this.idx < NeighborUpdater.UPDATE_ORDER.length; 
    }
}
```

NC更新的集体单位，代表五次NC更新。
一般围绕“更新核”发出NC更新，但排除某个方向时，使用这个单位。
---

这是相当值得注意的一点。当我们说“跳略器将人工栈塞满”时，这里“塞满”人工栈的，不是“更新本身”，而是这几个“更新单位（或者叫更新实例）”。例如，一次六向PP更新可以塞入6个单位，但一次六向NC更新却仅能塞入一个：每个PP更新都是独立的单位，而6次NC更新整体算作一个单位。
